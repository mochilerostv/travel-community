== Description ==

Mochileros Travel Deals Pro es un plugin que integra una comunidad premium de ofertas de viajes en tu sitio web de MochilerosTV. 

**Características principales:**

* 🌍 Comunidad premium de ofertas de viajes
* ✈️ Ofertas de vuelos con errores de tarifa
* 🏨 Chollos de hoteles verificados
* 🔔 Alertas personalizadas por aeropuerto
* 👥 Acceso a grupo privado de mochileros
* 🤖 Detección automática con IA
* 💰 Suscripción de solo €1,99/mes
* 📝 Registro de usuarios
* 🔒 Shortcodes de login/cuenta
* 📈 Dashboard de ofertas
* 🌐 Enlaces de afiliados configurables

== Frequently Asked Questions ==

= ¿Cómo funciona la suscripción premium? =

Los usuarios pagan €1,99/mes para acceder a ofertas exclusivas de viajes detectadas por nuestro sistema de IA.

= ¿Puedo personalizar el diseño? =

Sí, puedes modificar los estilos CSS desde el archivo `assets/mochileros-deals.css`.

= ¿Es compatible con mi tema? =

El plugin está diseñado para ser compatible con la mayoría de temas de WordPress.

== Shortcodes ==

* `[mochileros_register redirect="/comunidad"]` - Muestra el formulario de registro de usuarios
* `[mochileros_login redirect="/comunidad"]` - Muestra el formulario de inicio de sesión de usuarios
* `[mochileros_account]` - Muestra el panel de cuenta del usuario
* `[mochileros_deals_signup]` - Muestra el formulario de registro premium
* `[mochileros_deals_dashboard]` - Dashboard con ofertas activas
* `[mochileros_deals_premium]Contenido exclusivo[/mochileros_deals_premium]` - Protege contenido premium
* `[deal_link partner="booking" city="Roma" checkin="2025-09-10" checkout="2025-09-15"]Ver hotel[/deal_link]` - Genera enlaces de afiliados para ofertas de hoteles
